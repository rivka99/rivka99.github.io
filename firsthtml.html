MY Girls Who Code Experience <3

I am Rivka, a high school student who loves to code. I go to Bnos Chomesh Academy, which is a Jewish girls high school in Brooklyn. Being at Girls Who Code was one of the greatest and eye opening experiences of my life.

Girls Who Code is a really awesome organization you can check it out if you click the logo below.

My GWC projects

There were many projects I did throughout the duration of the program, I listed a few of them below.

  • Runner Game: a game that required a code that allowed an image to follow the curser around the screen, and collect objects to pick up points. I coded this game using python.
  • Obamicon project: In this project we used a picture from Obama's campaign and created a code that changed another picture to have the same color effect.
  • Robot Dance: This was one of the projects in which we programmed actual robots. We used code to make the robot move in different directions and we timed it to follow the beat of a song.
  • Mad Libs: This was one of the simpler projects, but it was really fun to create and play. I used a lot of variables to make this game, and when playing it I got some funny results.
